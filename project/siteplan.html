Website Planning Document

Site Name

Prato do Dia CV

The name “Prato do Dia CV” represents a local food service in Cape Verde that offers a fresh and affordable daily meal. It is a familiar expression in Portuguese-speaking cultures and clearly communicates the main purpose of the website.

The name was chosen because it is simple, memorable, and suitable for both an academic project and a real business application.

Optional domain: pratododiacv.com

Site Purpose

The purpose of this website is to provide information about a local food business that offers a daily meal service. The site will display the daily menu, prices, business hours, contact information, and ordering instructions. It will also help customers quickly see what is available each day.

Scenarios

  • What is today’s menu and how much does it cost?
  • How can I contact or place an order with the business?
