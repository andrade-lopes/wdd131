## 📘 Project Description 

This website was developed as a final assessment for the **WDD131** course. 
The objective is to demonstrate practical mastery of **HTML, CSS, and JavaScript ** 
through an informative website about **Mindelo, Cape Verde**, featuring responsive design and interactive features.

## 🌐 Site Structure

- `index.html` – Home page with information about the city  
- `about.html` – History and Culture of Mindelo  
- `contact.html` – Contact form with validation  
- `references.html` – List of sources used in the content  

---

## 🛠️ Technologies Used

- Semantic and accessible HTML5  
- CSS3 with media queries for responsive design  
- Modern JavaScript (ES6+) with:
  - DOM manipulation 
  - Events
  - Objects and Arrays
  - Template literals
  - Use of `localStorage`

---

## 📦 Implemented JavaScript Features

- Dynamic content rendering with template literals  
- Data storage in the browser with `localStorage`  
- Interaction with the DOM (dynamic display of places and messages)  
- Manipulation of arrays and objects  
- Reactions to events such as `click` and `DOMContentLoaded`
