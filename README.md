# WDD 131 – Dynamic Web Fundamentals

## Description
This repository contains the initial project for the **WDD 131** course, focused on web development fundamentals, including basic HTML, CSS, and JavaScript.

The goal of this project is to create a clean and functional website structure, ready for expansion and practice exercises from the course.
